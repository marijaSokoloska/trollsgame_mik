Играта ChatGuard е едноставен едукативен прототип што работи целосно во веб-прелистувач и не бара инсталација на сервер, 
база на податоци или дополнителни алатки. 

За да се стартува играта, потребен е компјутер со модерен прелистувач (Chrome, Edge или Firefox). 
Најпрепорачаниот начин за стартување е преку Visual Studio Code со екстензијата Live Server, бидејќи така сите ресурси (слики и скрипти) 
се вчитуваат правилно. 
По клонирање или преземање на проектот од Git репозиториумот, проектот треба да се отвори во Visual Studio Code, 
а потоа со десен клик на фајлот index.html да се избере опцијата „Open with Live Server“. 
Играта автоматски ќе се отвори во прелистувач и е подготвена за користење.

Играта може да се стартува и без Visual Studio Code, со директно отворање на фајлот index.html во прелистувач, 
но во тој случај можно е некои слики или функционалности да не се прикажуваат правилно, па затоа овој начин не е препорачан. 

Проектот работи целосно локално и не испраќа никакви податоци на интернет. Напредокот на играчот се чува во локалната меморија на прелистувачот (localStorage), 
што значи дека отклучените нивоа остануваат достапни и по освежување на страницата, сè додека не се избрише кешот на прелистувачот.
